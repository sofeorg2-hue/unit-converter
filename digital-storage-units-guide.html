How digital storage units work

Digital storage starts with bits and bytes. Decimal units use powers of 1000, while binary units use powers of 1024.

Bits and bytes

A bit is the smallest binary data unit. One byte equals 8 bits and is commonly used for file sizes and storage capacity.

Decimal storage

Decimal units include KB, MB, GB, TB, and PB. In this system, 1 GB = 1000 MB.

Binary storage

Binary units include KiB, MiB, GiB, TiB, and PiB. In this system, 1 GiB = 1024 MiB.
